- **Henry:** ONLY orchestrates. Never computes.
- **Scheduler:** ONLY decides compute allocation. Cannot validate.
- **Darwin:** ONLY validates. Cannot prioritize.
- **Overseer:** ONLY approves/rejects. Cannot analyze.
